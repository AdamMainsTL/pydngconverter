PyDngConverter¶

A Python Interface to Adobe’s DNG Converter.

Uses multiprocessing and Image manipulation to significantly speed up the process of convering RAW images to DNG.

class pydngconverter.DNGConverter(source, dest=None, compressed=<Compression.YES: 'c'>, camera_raw=<CRawCompat.ELEVEN_TWO: 11.2>, dng_version=<DNGVersion.ONE_FOUR: 1.4>, jpeg_preview=<JPEGPreview.MEDIUM: 1>, fast_load=False, linear=False, multiprocess=True, ray_args={})¶

Python Interface to Adobe DNG Converter

Takes the same parameters as the Adobe DNG Converter, but uses enums and such to make it easier to use.

Parameters

  • source (os.Pathlike) – Path to source directory containing raw files

  • dest (os.Pathlike, optional) – Path to destination. Defaults to source directory.

  • compressed (Compression, optional) – Enable DNG Compression. Defaults to Compression.YES.

  • camera_raw (CRawCompat, optional) – Set Camera Raw Compat vers. Defaults to CRawCompat.latest().

  • dng_version (DNGVersion, optional) – Set DNG Backwards Compat vers. Defaults to DNGVersion.latest().

  • jpeg_preview (JPEGPreview, optional) – JPEG Preview size. Defaults to JPEGPreview.MEDIUM.

  • fast_load (bool, optional) – Embed fast load data (for jpeg preview). Defaults to False.

  • linear (bool, optional) – Output linear DNG files. Defaults to False.

  • multiprocess (bool, optional) – Convert multiple files at once. Defaults to True.

  • ray_args (dict, optional) – Additonal args to pass to ray. Defaults to {}.

Raises

  • FileNotFoundError – DNG Converter cannot be found.

  • NotADirectoryError – Source directory does not exist or is not a directory.
